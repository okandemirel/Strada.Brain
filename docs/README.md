# Documentation Guide

Use these files as the authoritative source for current runtime behavior, environment variables, and security defaults:

- [README.md](../README.md)
- [SECURITY.md](../SECURITY.md)
- [src/config/README.md](../src/config/README.md)
- [src/channels/README.md](../src/channels/README.md)
- [src/dashboard/README.md](../src/dashboard/README.md)

The files under `docs/audit/`, `docs/remediation/`, and `docs/superpowers/` are historical audit snapshots, plans, and design documents. They record what was found or intended at a point in time, but they are not the source of truth for current runtime behavior or current environment variable defaults.
