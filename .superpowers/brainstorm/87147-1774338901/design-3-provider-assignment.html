Scoring Algoritması
function scoreProvider(provider, capabilityProfile):
 
  // 1. Hard filter: zorunlu capability yoksa elendir
  for tag in capabilityProfile.primary:
    if provider.scores[tag] == 0: return -1 // vision=0 ama vision gerekli -> ELENDIR
 
  // 2. Weighted capability score
  capScore = 0
  for tag in capabilityProfile.primary:
    capScore += provider.scores[tag] * CAPABILITY_WEIGHT // 0.6
 
  // 3. Preference score
  prefScore = provider.scores[capabilityProfile.preference] * PREFERENCE_WEIGHT // 0.3
 
  // 4. History bonus (execution trace'den)
  historyScore = getSuccessRate(provider, tag) * HISTORY_WEIGHT // 0.1
 
  return capScore + prefScore + historyScore

Assignment Kurallari
Hard Rules (ihlal edilemez)
1. Vision gerektiren node, vision=0 provider'a atanamaz
2. Provider healthCheck fail ise atama yapilmaz
3. Provider rate limit'e yakinsa baska provider tercih et
4. Kullanici hard-pin ise tum node'lar o provider'a
Soft Rules (optimization)
1. Diversity: 1 provider'a %60'tan fazla node atama
2. Cost balance: toplam maliyet bütçe limitinde kal
3. History: gecmiste basarili olan provider'i tercih et
4. Affinity: bagimliliktaki node'lari ayni provider'a at
