Web portal not built

The full web interface requires Node.js 20.19+ to compile. Your current Node.js version is older, so the portal was skipped during setup.

Required: Node.js 20.19+ or 22.12+
Installed: run node -v to check

Option 1 — Use terminal setup now

Configure Strada without building the portal:

# macOS / Linux
./strada setup --terminal

# Windows PowerShell
.\strada.ps1 setup --terminal

Option 2 — Upgrade Node.js, then rebuild

# macOS / Linux with nvm
nvm install 22 && nvm use 22

# Windows PowerShell with nvm-windows
nvm install 22.12.0 && nvm use 22.12.0

# or Windows PowerShell with winget
winget install OpenJS.NodeJS.LTS

# then run web setup
./strada setup --web
.\strada.ps1 setup --web

Once configured, start Strada with ./strada start on macOS/Linux or .\strada.ps1 start on Windows. The full portal becomes available after a successful build.
